# Number Game

A simple Java number guessing game where you have to guess a random number between 1-100.

## Features
- Random number generation (1-100)
- 10 attempts per round
- Score tracking system
- Multiple rounds
- Input validation and error handling

## How to Run

1. Make sure you have Java installed
2. Compile: `javac DecodeLabs_Java_P1.java`
3. Run: `java DecodeLabs_Java_P1`

## How to Play
- Guess a number between 1-100
- Get hints if your guess is too high or too low
- Try to guess in fewer attempts for more points
- Play multiple rounds to increase your score

## Requirements
- Java 8 or higher
- Git (for version control)
